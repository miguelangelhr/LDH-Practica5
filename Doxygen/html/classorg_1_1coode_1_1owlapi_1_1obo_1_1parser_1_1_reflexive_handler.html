Handles a tag. This is called by the OBOConsumer during parsing to handle tags that match the value returned by the getTagName() method.

Parámetros
currentIdThe id of the current frame.
valueThe value of the tag
qualifierBlock
commentThe hidden comment. This is made up of any characters between ! and the end of line.
