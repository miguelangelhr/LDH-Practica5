O getDefaultReturnValue (OWLObject object)
 

Descripción detallada

Author: Matthew Horridge
The University Of Manchester
Information Management Group
Date: 29-Jul-2008

Provides a default implementation of OWLObjectVisitorEx. Only the methods that need specific client implementation need be overridden. The adapter can be set up to return a default value.

O org.semanticweb.owlapi.util.OWLObjectVisitorExAdapter< O >.getDefaultReturnValue ( OWLObject  object)
protected

Gets the default return value for this visitor. By default, the default is null, but a fixed value (independent of the specified OWLObject object) can be specified in the constructor org.semanticweb.owlapi.util.OWLObjectVisitorExAdapter#OWLObjectVisitorExAdapter(Object)

Parámetros
objectThe object that was visited.
Devuelve
The default return value
