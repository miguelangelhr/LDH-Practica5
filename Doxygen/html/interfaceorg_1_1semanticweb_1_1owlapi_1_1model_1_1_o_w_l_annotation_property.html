Set<OWLAnnotationProperty> org.semanticweb.owlapi.model.OWLAnnotationProperty.getSubProperties ( OWLOntology  ontology)

Gets the annotation properties which are asserted to be sub-properties of this annotation property in the specified ontology.

Parámetros
ontologyThe ontology to be examined for SubAnnotationPropertyOf axioms.
Devuelve
A set of annotation properties such that for each property p in the set, it is the case that ontology contains an SubAnnotationPropertyOf(p, this) axiom where this refers to this annotation property.

Set<OWLAnnotationProperty> org.semanticweb.owlapi.model.OWLAnnotationProperty.getSubProperties ( OWLOntology  ontology,
boolean  includeImportsClosure 
)

Gets the annotation properties which are asserted to be sub-properties of this annotation property in the specified ontology and potentially its imports closure.

Parámetros
ontologyThe ontology to be examined for SubAnnotationPropertyOf axioms.
includeImportsClosureif true then the imports closure of this ontology is searched for SubAnnotationPropertyOf axioms that assert this property is the super property of some other annotation property. If false then only ontology is searched.
Devuelve
If includeImportsClosure is true, a set of annotation properties such that for each property p in the set, it is the case that at least one ontology in the imports closure contains a SubAnnotationPropertyOf(p, this) axiom where this refers to this annotation property. If includeImportsClosure is false, a set of annotation properties such that for each property p in the set, it is the case that ontology contains a SubAnnotationPropertyOf(p, this) axiom where this refers to this annotation property.

boolean org.semanticweb.owlapi.model.OWLAnnotationProperty.isDeprecated ( )

Determines if this annotation property has an IRI corresponding to owl:deprecated. An annotation along the owl:deprecated property which has a value of "true"^^xsd:boolean can be used to deprecate IRIs. (See http://www.w3.org/TR/2009/REC-owl2-syntax-20091027/#Annotation_Properties of the OWL 2 specification.

Devuelve
true if the IRI of this annotation property is owl:deprecated, where owl: expands to the usual prefix, otherwise false.
