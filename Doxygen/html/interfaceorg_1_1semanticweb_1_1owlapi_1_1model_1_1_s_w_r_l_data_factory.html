SWRLBuiltInAtom org.semanticweb.owlapi.model.SWRLDataFactory.getSWRLBuiltInAtom ( IRI  builtInIRI,
List< SWRLDArgument >  args 
)

Creates a SWRL Built-In atom. Builtins have predicates that are identified by IRIs. SWRL provides a core set of builtins, which are described in the OWL API by the org.semanticweb.owlapi.vocab.SWRLBuiltInsVocabulary.

Parámetros
builtInIRIThe builtin predicate IRI
argsA non-empty set of SWRL Arguments.
Excepciones
IllegalArgumentExceptionif the list of arguments is empty
Devuelve
A SWRLBuiltInAtom whose predicate is identified by the specified builtInIRI and that has the specified arguments
